# student-grades-manager
  A .py program that can create a new student and input name, Birth year, Science grade, Math grade, English grade by the user, and it calculates the percentage, it can show student info, rank students, get the top student, and remove a student. It is powered by Json file as a database file

    • Features:
        ○ Create a student.
        ○ Remove a student
        ○ Save student name, Grades(English, math, science), and birth year and calculate the percentage.
        ○ Show student info.
        ○ Rank all students according to their percentages.
        ○ Json database that saves all the data.
        ○ Clear database.

    • How to use:
        ○ Run the main.py file.
